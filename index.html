Welcome to Melbourne Transit Pulse—a live map showing Melbourne's trains moving in real time. This app uses live public transport data from Transport Victoria to show how the train network moves throughout the day. The data is updated every 30 seconds.

HOW TO USE
🔍 Zoom to view stations.
🚆 Click a train for live details.
